ShopUpNow - Application de vente de tickets de théâtre en ligne

ShopUpNow est une application web innovante permettant aux utilisateurs d'acheter des tickets de théâtre pour des événements en ligne. Le projet a été conçu pour faciliter la réservation de billets en permettant aux utilisateurs de découvrir des spectacles, de consulter les horaires, et d'acheter des tickets en quelques clics.

Le système est équipé d'un moteur de géolocalisation pour permettre aux utilisateurs de trouver des événements proches de leur position, ainsi que de Thymeleaf, Spring Boot pour le back-end et HTML/CSS/JS pour le front-end, avec une base de données MySQL pour gérer les événements, les utilisateurs et les transactions.

Fonctionnalités principales :

  • Affichage des événements : Visualisation des spectacles, lieux et horaires disponibles.
  • Recherche par localisation : Trouver les événements proches via la géolocalisation.
  • Réservation de tickets : Achat facile de billets avec paiement sécurisé.
  • Gestion des utilisateurs : Création de comptes utilisateurs, gestion des profils et des réservations.
  • Interface intuitive : Un design moderne et réactif avec Bootstrap et une navigation simple.

Technologies utilisées :

  • Back-end : Spring Boot
  • Template Engine : Thymeleaf
  • Base de données : MySQL
  • Frontend : HTML5, CSS3, JavaScript, Bootstrap
  • Fonctionnalité spéciale : Géolocalisation des événements

Cette application a été développée en mettant l’accent sur une expérience utilisateur fluide et intuitive. Le design est responsive, permettant à l'utilisateur de réserver des tickets aussi bien depuis un bureau que sur un appareil mobile.
